# Focused Gov

Focused Gov is an AI-powered government exam preparation platform designed to help aspirants prepare smarter, stay consistent, and improve performance through focused learning tools.

Live Demo: https://focused-gov.vercel.app/

---

## Overview

Focused Gov combines AI-driven preparation tools with modern learning workflows for competitive government exams.

The platform is designed for aspirants preparing for:
- UPSC
- SSC
- Banking
- Railways
- State PCS
- Defence
- Other government recruitment exams

Focused Gov focuses on:
- Personalized preparation
- Smart revision
- Performance analytics
- AI-assisted learning
- Structured study systems
